# 🎓 PlacementHub V1 - Academic Placement Portal

PlacementHub is a high-end, production-ready platform designed to bridge the gap between academic excellence and industry leadership. It streamlines the recruitment process for Students, Recruiters, and Administrators with a premium, glassmorphism-inspired user interface.

## 🚀 Features

- **Multi-Role Dashboards:** Tailored experiences for Students, Recruiter, and Admin.
- **Instant Sync Engine:** Real-time job posting and application tracking.
- **Secure Profile Vault:** Enterprise-grade resume hosting and data protection.
- **Premium UI/UX:** Advanced CSS techniques including glassmorphism and smooth micro-interactions.
- **Application Lifecycle:** Full transparency from submission to hire.

## 🛠️ Tech Stack

- **Backend:** Python / Flask
- **Database:** SQLite (Local persistent storage)
- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript
- **Deployment:** Vercel-ready

## 📦 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Awesomedisha/Acaedmic_Portal_MAD.git
   cd Acaedmic_Portal_MAD
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Initialize the database:**
   ```bash
   python database/init_db.py
   ```

4. **Run the application:**
   ```bash
   python app.py
   ```
